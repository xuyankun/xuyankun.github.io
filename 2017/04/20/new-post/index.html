A New Start

发表于 2017-04-20 | 分类于 Blog | 阅读量

Abstract：
这是我最近刚刚开通的个人主页，也算是一个博客，但值得欣喜的是这个主页是开源搭建的，域名也用上了自己的名字。这是我的第一篇文章，希望是一个新的开始 …

正文

最近开通了 Github ，这是目前世界最庞大的一个开源世界，从功能上讲，是一个带有简单社交功能的云端代码仓库服务，现在很多的软件都需要通过此平台安装下载。我不是程序员，使用这个平台是希望通过此平台学习和分享，我想，学习他人的知识和分享自己的知识或许也是开源世界的初衷。

Github 只要去网站开通就可以了。不过在搜索资料的同时，发现 Github 居然还可以利用 Github pages 来搭建免费个人主页，这真是太良心了！可我又没学过 html，所以只好再借鉴前人的智慧，结合 hexo 来搭建了现在所看到的个人主页，主题利用的是 Next ，看起来确实挺高大上，我也很满意。可能会在之后的文章里详细介绍一下如何搭建这个主页。

我小时候，大概是初中那会就觉得自己整天有一肚子话没地方说，就找了当时各种门户网站开通博客，那时还真玩了好一会，网站到现在还在那，有时看看觉得那个时候的我好青涩啊。当时每次写博文时都会觉得自己说出了什么真理，悟出了什么哲理，可一个星期再打开写新的文章时就已经发现上个星期写的那叫什么玩意儿啊，哈哈！但不过现在想想看，我也正是从那个时候开始思考了许多同龄人很来后才思考的问题，我们通常称这个现象叫早熟。虽然大部分早期的早熟都体现为思春，也就是喜欢上班里的小姑娘了，可又由于人家对你爱答不理，于是进行了人生初期对于“孤独”、“寂寞”等命题的思考，所以等到了大学之后再看到同学发出什么人生寂寞等哀怨时，我只会微微一笑，在内心说一句“Naive”。我猜测正是这样的一种早期开始对人生的“思考”，奠定了我现如今给人没事总爱瞎BB的感觉，其实我一直都是不爱多说话的人设啊喂！

这次开通了个人的主页，既圆了我又可以开始写博客的心愿，也帮我实现了自己搭建主页的夙愿（虽然都是傻瓜式的！）。这应该算是一个新的开始吧（虽然我在我每个开通就没使用过的博客的第一篇文章也是这么写的…），希望自己，也督促自己能坚持下去。有想法就写写想法，有学习笔记就写点学习笔记，有代码就写点代码，反正就这么下去吧。

多矫情一句，开始做这些事的动力来源于现在留学期间的一个同班同学，我们一起学习期间他的行为让我很受鼓舞，十分感谢
